Commands

Authentication

  • /auth login - Authenticate with Schwab API
  • /auth logout - Log out from Schwab API
  • /auth status - Check authentication status

Stock Quotes

  • /quote <symbol> - Get a stock quote

Watchlists

  • /watchlist create <name> [symbols] - Create a new watchlist
  • /watchlist list - List all watchlists
  • /watchlist add <name> <symbol> - Add a symbol to a watchlist

Price Alerts

  • /alert create <symbol> <price> <condition> - Set a price alert

Streaming & Dashboards

  • /stream start <symbol> - Start real-time streaming
  • /dashboard create [symbols] - Create a stock dashboard

Paper Trading

  • /trade buy <symbol> <quantity> - Buy stocks/options
  • /trade sell <symbol> <quantity> - Sell stocks/options
  • /trade history - View trade history
